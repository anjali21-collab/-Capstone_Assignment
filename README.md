## Description
This project implements a Student Management System in Java. It allows you to add, view, search, delete, and sort student records, and also save & load them from a file.

## Usage
1. Clone the repository  
2. Compile and run the `App.java` file  
3. Follow on-screen menus to perform student operations  
4. Save and exit when finished  

## Technologies
- Java  
- Text file for data storage  

## Repository Structure

| File / Folder | Description |
|---------------|-------------|
| App.java | Main application entry |
| model/Student.java | Student model class |
| service/StudentManager.java | Logic for managing student records |
| exception/StudentNotFoundException.java | Custom exception |
| util/Loader.java | Loading animation utility |
| students.txt | Data file for storing student records |
